Navigation Bar

Starting from the navigation menu, it contains every page that is required for the assignment. The title on the top left side of the navigation bar is a 'label' tag used to portray each page's title.The shadow effect used on navigation menu is used for the text to have a glowing 3d effect.

For a responsive navigation bar, it operates after the screen pixels gets below 801 widths.

The burger icon is made by a div using a 2px height, 18px width element to form a single dash and then using before sector and after sector to insert more element on the top and the bottom of the element to make it appear like a hamburger icon. The menu scrolls down without overlapping other elements.

However, the code is not my own, but I've learned it from a youtube tutorial titled "Responsive Pure CSS Menu Tutorial(No Javascript)" from the channel 'FollowAndrew' The video. [Accessed 29 May 2021]

The self-writing animation is the work of the keyframe used on the greeting text. The text was first positioned then been regulated with the syntax ‘animation’. The given name to the tag is ‘typing, blinking’ (typing being the self typing animation, blink is the insertion cursor animation.) On 2.5seconds, and in 22 steps, the writing animation completes itself. The blink is set to infinite, where it is set as a loop for 1second delays.

Contact page

Background image is from Here.[Accessed 19 May 2021]

Pressing the contact us button will open a pop-up form. The process started with making the form first and maintaining its position to the centre, then changing the display to none. A checkbox was used to display the form container while in a checked state; the container for the form is displayed while unchecked the display remains none for the container, so it doesn't appear on the screen. The id for the checkbox is 'show', and with the help of 'label' syntax for the id, I was able to customize the size for the "contact us" button, which is also the checkbox.

To do so, I had to learn the codes from a youtube video from CodingNepal.

Form action mailto is applied; submitting the form will open your default mail application. There is no way to redirect the message to us as I'm inexperienced in coding and limited to the languages.
